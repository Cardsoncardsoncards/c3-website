# Goddess of Victory Nikke Weiss Schwarz Card Guide (Australia)

Goddess of Victory: Nikke is a mobile gacha game known for its striking character designs and sci-fi setting, and its Weiss Schwarz release brought that distinctive art style directly onto the card game stage. Illustrations pull from the game's beautiful in game visuals alongside new artwork drawn specifically for this release, giving Nikke fans genuine crossover appeal between the mobile game and physical card collecting.

## The set at a glance

Goddess of Victory: Nikke is currently represented on this site by a single tracked release, released 18 April 2025, with a total of 376 cards. The set code is NIK, and the release featured 115 base card types plus 173 parallel versions, packed 8 cards per pack, 12 packs per display, 24 displays per carton. Note that Bushiroad has announced a NIKKE Vol.2 booster set scheduled for release on 25 September 2026, which will be a genuinely separate follow up product once it launches. As that set has not been released yet, it is not covered by this guide or the current database.

The Booster Pack releases alongside a Trial Deck+, and cards from both products share the same NIK card number prefix, meaning they combine freely into a single deck under Neo Standard construction rules.

## Deck building around your favourite Nikkes

Because Nikke as a game features a large roster of playable characters, each with their own personality and combat role, the Weiss Schwarz card pool reflects that same breadth. This structure suits players who already have favourite characters from the mobile game, since building a deck around your preferred squad carries the same appeal as team building within the game itself.

## Chase cards and signature pulls

Every pack carries a chance of ultra rare SP and even rarer SSP cards, featuring unique artwork and character signatures. These are genuinely prized items among Nikke collectors, both for their scarcity and for the specially commissioned artwork that accompanies them. Display boxes included one of sixteen exclusive Going First and Going Second cards in first edition print runs, and first edition cartons carried an English Edition exclusive stitched playmat featuring Nikke artwork.

Because this is a gacha game adaptation with a genuinely large existing player base, demand for top rarity cards featuring the most popular in game characters has remained strong since release.
