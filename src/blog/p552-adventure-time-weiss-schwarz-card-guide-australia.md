# Adventure Time in Weiss Schwarz: Finn, Jake and a Genuinely Modular Build

Adventure Time brings Finn the Human and Jake the Dog into Weiss Schwarz as a single, densely built set, and community set overviews confirm it as one of the more versatile properties in the game, supporting multiple genuinely different competitive builds from the same modular card pool. If you already play and want the archetype detail, jump ahead. If you are newer to Weiss Schwarz, the primer below covers the core rules, with the full mechanics glossary linked at the end.

## The set at a glance

Adventure Time is a single stand-alone set, released 20 November 2020, with a total of 178 cards. It has not received a second release since, so this remains the complete Adventure Time collection currently available.

## Quick primer: how Weiss Schwarz works

Every deck is 50 cards split across colours, cards get stronger as your clock (a face-down pile built from damage taken) grows, climax cards (up to 8 per deck) combo with character abilities on the right turn, and you can run up to four copies of any named card. For the full breakdown, read our [complete rules glossary](/blog/how-weiss-schwarz-works-rules-deck-building-australia).

## A Genuinely Modular, Multi-Archetype Set

Community deck analysis confirms Adventure Time's card pool leans yellow with strong red options also available, built around Finn, Jake, Princess Bubblegum, and Wildberry Princess. What makes this property stand out is that it is built around modular character and event building blocks that can be mixed and matched into several distinct, functional decks. Documented approaches include a Finn and Jake finisher build aiming to close games out with the confirmed level 3 "Finn the Human & Jake the Dog" combo card, a Wildberry Princess "choices" build that uses her repeated on-death effect to generate ongoing options, and an eight standby trigger build, though community analysis notes the standby approach does not receive quite as much dedicated support as the other two.

He is a confirmed general-purpose event-fetching character who can grab any level 1 or higher event card when played, tying together the set's broader event support.

Weiss Schwarz does not use a rotating standard format, so cards generally remain playable, though always check a specific event's rules before entering.
