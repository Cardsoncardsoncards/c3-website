## Disney Lorcana in Australia: Which Set to Buy?

Disney Lorcana launched in late 2023 and has become one of the three strongest-selling TCGs at Australian retail behind Pokemon and MTG. The Disney IP drives collector demand that goes well beyond the game's competitive playerbase — parents, Disney fans, and casual collectors are all buying it.

This guide covers every Lorcana set available in Australia, ranked for new buyers.

## The Sets in Order

**The First Chapter (TFC):** Set 1. The Elsa "Let It Go" enchanted card is the headline chase card and remains one of the most valuable singles in the game. Heavily reprinted but still holds demand due to collector status.

**Rise of the Floodborn (ROTF):** Set 2. Strong gameplay additions. The Amber/Sapphire archetype emerged here. Solid sealed value.

**Into the Inklands (ITI):** Set 3. Treasure Planet and Atlantis characters. Less mainstream Disney recognition but strong artwork quality.

**Ursula's Return (UR):** Set 4. Ursula-focused villain set. High demand from villain collectors.

**Shimmering Skies (SS):** Set 5. Sky and weather themed. Current meta staples.

**Azurite Sea (AS):** Set 6. Nautical theme. Most recent set at time of writing.

## The Enchanted Chase

Every Lorcana set has Enchanted cards — alternative art versions of key characters with a shimmering foil treatment. These are the primary collector targets. Pull rate is approximately 1 per box, sometimes slightly better.

The Elsa Enchanted from The First Chapter remains the highest value single. It trades at AU$200 to $400 depending on condition and timing.
