# Hatsune Miku in Weiss Schwarz: Two Different Games, One Property

Hatsune Miku's Weiss Schwarz catalogue actually spans two genuinely different source games sharing the same virtual idol, and keeping them straight matters more here than it might first appear. The original Project DIVA rhythm game material sits in yellow and green, while the newer Colorful Stage mobile rhythm game brings its own separate cast and card pool under the same property banner. If you already play and want the breakdown between the two, jump ahead. If you are newer to Weiss Schwarz, the primer below covers the core rules, with the full mechanics glossary linked at the end.

## The set at a glance

Hatsune Miku spans 3 tracked releases, Project DIVA f (15 November 2013), Project DIVA F 2nd (February 2015), and HATSUNE MIKU: COLORFUL STAGE! (June 2025), totalling 729 cards.

## Quick primer: how Weiss Schwarz works

Every deck is 50 cards split across colours, cards get stronger as your clock (a face-down pile built from damage taken) grows, climax cards (up to 8 per deck) combo with character abilities on the right turn, and you can run up to four copies of any named card. For the full breakdown, read our [complete rules glossary](/blog/how-weiss-schwarz-works-rules-deck-building-australia).

## Two Different Games Under One Property

This is the property where checking the actual source material matters most before buying or building. The original Project DIVA trial deck material is confirmed as yellow and green, while COLORFUL STAGE! is tied to a completely different mobile rhythm game with its own cast of human characters alongside the virtual idols. Treating these as fully interchangeable is a real risk, similar to the Fate/Zero and Prisma Illya distinction covered elsewhere in this series. A card from the Project DIVA era reflects the older rhythm game's presentation and colour identity, while Colorful Stage material draws on a separate, newer game with its own character roster. Since the property spans two source games, the four copy limit reinforces deciding which game's card pool to build from as the first real decision here.

## Frequently Asked Questions

**Are Project DIVA and Colorful Stage the same game?**
No. Project DIVA is the original rhythm game the earlier trial deck material is built around, while Colorful Stage is a separate, newer mobile rhythm game with its own cast, both under the same Hatsune Miku property banner.

**What colours does the original Project DIVA trial deck use?**
Yellow and green, giving new players a defined starting pairing built around Miku and the Project DIVA aesthetic.

**Is there crossover synergy between Project DIVA cards and Colorful Stage cards?**
This has not been confirmed, given they come from different source games with different casts, treat them as separate collecting and deck building lanes unless you can confirm otherwise for a specific card.

**Do older Hatsune Miku sets remain legal in tournament play?**
Weiss Schwarz does not use a rotating standard format, so older cards generally remain playable, though always check a specific event's rules before entering.

**Should I buy Project DIVA cards or Colorful Stage cards as a new collector?**
That depends on which game you are actually a fan of. Check which release a card comes from before buying, since the two are not interchangeable despite sharing the Hatsune Miku name.

**How does the four copy limit interact with a property spanning two different games?**
It reinforces committing to one game's card pool rather than trying to build a single deck that mixes cards from two source games that were not necessarily designed to combine.

## What to check before buying

1. Confirm which of the two source games a listing belongs to before buying, Project DIVA or Colorful Stage, since the character rosters and colour identities differ.
2. Check whether a card is base rarity or a signed variant before paying a premium, since rarity structures differ release to release.
3. If deck building specifically, check for confirmed cross-compatibility between the two games before assuming a mixed build is viable, since this has not been established either way.
