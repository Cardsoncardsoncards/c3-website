# Disgaea in Weiss Schwarz: Prinnies, Netherworld and a Genuine Rule Exception

Disgaea brings Laharl, Etna, and Flonne into Weiss Schwarz in a documented red and green colour identity, and its card pool contains a genuinely rare exception to one of the game's core deck building rules that is worth knowing before you buy in. If you already play and want the specifics, jump ahead. If you are newer to Weiss Schwarz, the primer below covers the core rules, with the full mechanics glossary linked at the end.

## The set at a glance

Disgaea has 2 tracked releases, an original Trial Deck (30 May 2009) and a full booster set (24 June 2016), totalling 219 cards. A note on the older entry: our database currently shows only 1 card recorded for the 2009 Trial Deck, which is almost certainly incomplete legacy data rather than the true size of that release, so treat the 2016 set's 218 cards as the confirmed, reliable core of this property's card pool.

## Quick primer: how Weiss Schwarz works

Every deck is 50 cards split across colours, cards get stronger as your clock (a face-down pile built from damage taken) grows, climax cards (up to 8 per deck) combo with character abilities on the right turn, and you can run up to four copies of any named card. For the full breakdown, read our [complete rules glossary](/blog/how-weiss-schwarz-works-rules-deck-building-australia).

## Prinnies, Angels, and the Rule-Breaking Exception

Disgaea's trial deck material is confirmed as mainly red and green, built around Laharl and the wider Netherworld cast. The confirmed card pool builds around 《Prinny》 and 《Angel》 character traits, and here is the standout feature: certain Prinny characters carry text explicitly stating you can include any number of cards with that same name in your deck, breaking the usual four copy cap for that specific card. This is a real, documented mechanic tied directly to the disposable, mass-produced nature of Prinnies in the source material, and one of the few properties in the whole catalogue where the standard four copy rule genuinely does not apply to every card. It is worth double checking the exact wording on any specific Prinny card before assuming the exception applies, since the unlimited-copy text is tied to specific confirmed cards rather than the Prinny trait as a whole.

## Frequently Asked Questions

**Is it true Disgaea has an exception to the four copy limit?**
Yes, confirmed via card text, certain Prinny characters explicitly state you can include any number of cards with that same name in your deck, a genuine documented exception rather than a misprint.

**What are the Prinny and Angel traits for?**
They are confirmed character traits that certain card abilities check for or search the deck by, Prinny characters in particular have documented combo support tied to specific climax cards.

**Can I build a Prinny swarm deck using the copy exception?**
The confirmed unlimited-copy text on certain Prinny cards genuinely supports a swarm-style approach that the standard four copy limit would not normally allow.

**Do older Disgaea sets remain legal in tournament play?**
Weiss Schwarz does not use a rotating standard format, so older cards generally remain playable, though always check a specific event's rules before entering.

**Does the four copy limit still apply to Laharl, Etna, and Flonne's own cards?**
Yes, the confirmed exception applies specifically to certain Prinny cards with the explicit unlimited-copy text, not to the named main characters, whose cards follow the standard four copy limit like any other Weiss Schwarz character.
